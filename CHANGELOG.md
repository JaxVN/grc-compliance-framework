## [1.0.0] — 2025-01-01

### Added
- Initial framework schema for StandardControl, TechnicalControl, Mapping, Evidence
- ISO 27001:2022 control reference structure
- NIST CSF 2.0 control reference structure
- ACSC Essential Eight reference structure
- Mapping templates: ISO 27001 ↔ Microsoft 365 Secure Score
- Data dictionary and methodology documentation
- CC BY 4.0 license
- CITATION.cff for GitHub/Zenodo citation support

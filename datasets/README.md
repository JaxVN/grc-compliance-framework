# Datasets

This folder contains versioned mapping datasets exported from the GRC framework.

## Format

Each dataset is a CSV or JSON snapshot corresponding to a GitHub release.

| File | Version | DOI | Description |
|---|---|---|---|
| _(coming with v1.0.0 release)_ | — | — | — |

## Usage

Import CSV datasets into any GRC tool, spreadsheet, or database.
JSON datasets follow the schema defined in `framework/schema/schema.json`.
